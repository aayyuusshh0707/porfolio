Netflix Yellow Project

As part of my academic journey, I designed and developed the Netflix Yellow Website, a streaming movie platform inspired by Netflix's design aesthetics. This project, built using HTML, CSS, and JavaScript, features a user-friendly interface for browsing and streaming movies. Key functionalities include: User Authentication: Implemented a secure login page with Firebase database connectivity, ensuring seamless access to user accounts and personalized content preferences. Front-End Development: Showcased proficiency in HTML, CSS, and JavaScript, creating an intuitive and visually appealing user interface. Database Integration: Integrated Firebase for robust data management and user authentication. This project highlights my skills in front-end development, user experience design, and database integration, demonstrating my ability to create comprehensive web applications.
